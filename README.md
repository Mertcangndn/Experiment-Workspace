# Experiment Workspace 🔬

Welcome to my personal engineering sandbox! This repository serves as a consolidated workspace where I test new frameworks, build experimental mini-projects, and deepen my understanding of specific technologies. 

Instead of cluttering my profile with dozens of small, isolated repositories, I use this space as a "development laboratory" to house active learning projects and prototypes.

## 🗂️ Workspace Structure

This monorepo currently contains the following experimental environments:

### 1. `ROS2-Workspace` 🤖
A dedicated environment for my **Robot Operating System 2 (ROS 2)** studies. This directory contains custom nodes, subscriber/publisher setups, and simulations aimed at understanding autonomous systems and robotics middleware.

### 2. `OpenCV-Workspace` 👁️
My playground for **Computer Vision** using Python and OpenCV. This section includes scripts for image processing, feature detection, and algorithmic visual analysis.

### 3. `RestAPI-Workspace` 🌐
A backend development environment focused on designing, testing, and implementing **RESTful APIs**. This includes routing, database connections, and handling HTTP requests/responses (primarily utilizing Java-based architectures).

---
*Note: This repository is structured as a monorepo, constructed using `git subtree` to merge multiple independent project repositories while fully preserving their original commit histories.*
